### Installation

Follow these steps to get started with **faststack mcp toolkit:**

**Step 1**: Install the sdk:

<CodeGroup>

```bash npm
npm i @faststack.ai/mcp
```


```bash yarn
yarn add @faststack.ai/mcp
```


```bash bun
bun add @faststack.ai/mcp
```


```bash pnpm
pnpm install @faststack.ai/mcp
```

</CodeGroup>

**Step 2**: Navigate to the docs directory (where the `docs.json` file is located) and execute the following command:

```bash
mintlify dev
```

A local preview of your documentation will be available at `http://localhost:3000`.
